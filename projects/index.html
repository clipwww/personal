專案經歷 Projects ​

這裡記錄了我在不同公司與專案中的開發經歷，涵蓋官方網站、後台管理系統、行銷活動、金流整合等多種類型。

專案分類 ​

RE 紅包 / RE·X 點數魔術師 ​

RE·X 是一個點數回饋平台，我參與了多個核心專案的開發：

  • RE·X 官方網站 - SSR 網站、SEO 優化、多語系
  • RE·X BOSS 店家管理系統 - Hybrid App、金流串接、資料視覺化
  • RE·X Admin - 企業內部管理後台
  • RE·X Web SDK - Apple Pay / Google Pay on Web
  • LINE@ LIFF - LINE 前端框架整合
  • 多項行銷活動小遊戲 - PixiJS 遊戲開發

FunNow ​

FunNow 是即時預訂平台，我負責維護與開發店家端後台管理系統。

蝦米智慧媒體 WhatsMedia ​

在 WhatsMedia 期間開發的百貨官網、商場系統等專案。

SOHO 接案 ​

獨立接案時期完成的專案，包含格上租車 APP 小遊戲、GoShare 行銷活動等。
